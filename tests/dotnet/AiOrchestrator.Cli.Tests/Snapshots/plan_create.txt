aio plan create

Create a new plan in SCAFFOLDING state.

Usage: aio plan create [options]

Options:
  --json        Emit machine-readable JSON output.
  --no-color    Disable ANSI color output (also respects NO_COLOR env var).
  --help        Show this help.
  --name <name>  Human-readable plan name (required).
  --store <path> Plan store root; validated via IPathValidator.

Exit codes:
  0   OK
  64  Usage error
  65  Config error
  74  I/O error
  77  Permission denied
  80  Plan failed
  81  Plan canceled
  82  Plan partial
  90  Daemon unavailable
  99  Internal error
