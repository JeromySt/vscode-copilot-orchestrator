# OE0003 — Microsoft.VisualStudio reference outside extension transport

**Severity:** Error  
**Category:** AiOrchestrator  

## Rule description

`Microsoft.VisualStudio.*` APIs are strongly coupled to the VS SDK and must be confined to the
VS extension transport project. Business logic that imports Visual Studio packages cannot be
unit-tested without the VS SDK host, and it violates the adapter pattern boundary.

## Non-compliant code

```csharp
// ❌ Business logic importing VS SDK
using Microsoft.VisualStudio.Shell; // OE0003

namespace AiOrchestrator.Core
{
    public class Worker { }
}
```

## Compliant code

```csharp
// ✅ Only the VS transport project may use VS APIs
namespace AiOrchestrator.VsExtension
{
    using Microsoft.VisualStudio.Shell;
    // Adapter implementation here
}
```

## Suppression

Only suppressible in projects that ARE the VS extension transport. Add a project-level suppression.
