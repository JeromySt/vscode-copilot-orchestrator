# OE0004 — System.IO.File/Directory used outside FileSystem project

**Severity:** Error  
**Category:** AiOrchestrator  

## Rule description

Direct use of `System.IO.File` or `System.IO.Directory` is allowed only in the
`AiOrchestrator.FileSystem` project (the adapter layer). All other projects must go through the
`IFileSystem` interface to keep file I/O mockable and testable.

## Non-compliant code

```csharp
// ❌ Direct file access outside the FileSystem adapter
using System.IO;
namespace AiOrchestrator.Core
{
    public class ConfigLoader
    {
        public string Load(string path) => File.ReadAllText(path); // OE0004
    }
}
```

## Compliant code

```csharp
// ✅ Use the IFileSystem abstraction
namespace AiOrchestrator.Core
{
    public class ConfigLoader
    {
        private readonly IFileSystem _fs;
        public ConfigLoader(IFileSystem fs) => _fs = fs;
        public Task<string> LoadAsync(string path, CancellationToken ct)
            => _fs.ReadAllTextAsync(path, ct);
    }
}
```

## Suppression

Only suppress inside `AiOrchestrator.FileSystem` implementations.
