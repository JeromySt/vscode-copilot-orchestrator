# OE0006 — LibGit2Sharp or git shell-out outside AiOrchestrator.Git

**Severity:** Error  
**Category:** AiOrchestrator  

## Rule description

Git operations must be confined to the `AiOrchestrator.Git` project and exposed through the
`IGitOperations` interface. Using `LibGit2Sharp` or shelling out to `git` anywhere else leaks
git semantics into business logic and makes tests difficult.

## Non-compliant code

```csharp
// ❌ Using LibGit2Sharp outside the Git adapter
using LibGit2Sharp; // OE0006
namespace AiOrchestrator.Core
{
    public class RepoScanner { }
}
```

## Compliant code

```csharp
// ✅ Use IGitOperations from DI
namespace AiOrchestrator.Core
{
    public class RepoScanner
    {
        private readonly IGitOperations _git;
        public RepoScanner(IGitOperations git) => _git = git;
    }
}
```

## Suppression

Only suppress inside `AiOrchestrator.Git` project files.
