# OE0005 — System.Diagnostics.Process used outside Process project

**Severity:** Error  
**Category:** AiOrchestrator  

## Rule description

`System.Diagnostics.Process` is a low-level OS abstraction that must be wrapped by the
`AiOrchestrator.Process` project and exposed via `IProcessSpawner`. Calling it directly
makes code untestable and bypasses sandboxing or monitoring hooks.

## Non-compliant code

```csharp
// ❌ Direct process management outside the Process adapter
using System.Diagnostics;
namespace AiOrchestrator.Core
{
    public class Launcher
    {
        public void Start() { var p = new Process(); } // OE0005
    }
}
```

## Compliant code

```csharp
// ✅ Use IProcessSpawner from DI
namespace AiOrchestrator.Core
{
    public class Launcher
    {
        private readonly IProcessSpawner _spawner;
        public Launcher(IProcessSpawner spawner) => _spawner = spawner;
        public Task<int> StartAsync(string exe, CancellationToken ct)
            => _spawner.SpawnAsync(exe, ct);
    }
}
```
