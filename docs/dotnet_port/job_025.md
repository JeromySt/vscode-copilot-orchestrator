# Job 025 — Agent Runners

**Tier:** T6
**Producer id:** `t6-agent`
**Dependencies:** `t2-process`, `t3-line-view`, `t3-event-bus`, `t6-output`, `t5-shell`
**Estimated LOC:** ~1,800 production, ~2,400 tests

## Goal

Agent runner family per §3.10.2 + §3.31 v1.1 fixes. One concrete runner per supported agent CLI:

- **ClaudeCodeRunner** (`claude`)
- **CodexCliRunner** (`codex`)
- **GeminiCliRunner** (`gemini`)
- **CopilotCliRunner** (`copilot`) — Microsoft Copilot for native CLI
- **GhCopilotRunner** (`gh copilot`) — GitHub Copilot CLI plugin (DISTINCT from CopilotCliRunner per v1.1 fix)
- **QwenRunner** (`qwen`)

Each implements the common handler set: `SessionIdHandler`, `StatsHandler`, `TaskCompleteHandler`, `ContextPressureHandler`.

    public ImmutableArray<string> AllowedUrls { get; init; } = ImmutableArray<string>.Empty;

    public int MaxTurns { get; init; } = 30;
    public bool ResumeSession { get; init; } = false;
    public ImmutableDictionary<string, string> Env { get; init; } = ImmutableDictionary<string, string>.Empty;

public enum Effort { Low, Medium, High, Xhigh }

public sealed record AgentRunResult { public required int ExitCode { get; init; } public required string? SessionId { get; init; } public required AgentStats Stats { get; init; } public required TimeSpan Duration { get; init; } public required bool TaskCompleteEmitted { get; init; } public required ImmutableArray<RepoRelativePath> ChangedFiles { get; init; } }
public sealed record AgentStats { public required int InputTokens { get; init; } public required int OutputTokens { get; init; } public required int Turns { get; init; } public required decimal? EstimatedCostUsd { get; init; } public required ImmutableDictionary<string, long> ProviderRaw { get; init; } }

### Behavioral invariants

1. **INV-1** — Each runner spawns its CLI via `IProcessSpawner` (NEVER `Process.Start`).
2. **INV-2** (RUNNER-DISTINCT v1.1) — `CopilotCliRunner` and `GhCopilotRunner` are distinct types and resolve to different executables (`copilot` vs `gh`); confusion test enforces.
3. **INV-3** — Each runner injects allowed folders / allowed URLs into the appropriate runner-specific flag (e.g. Claude `--allowed-tools`, Copilot `--allow-tool ShellExec(...)`).
4. **INV-4** — `SessionIdHandler` uses `[GeneratedRegex]` patterns; resumed sessions pass session ID via runner-specific resume flag.
5. **INV-5** — `StatsHandler` produces non-null `AgentStats` even on partial parse (zeros + `ProviderRaw`).
6. **INV-6** — `TaskCompleteHandler` flips `TaskCompleteEmitted` exactly once; absence at process exit is a non-fatal warning event.
7. **INV-7** — `ContextPressureHandler` thresholds: Rising 0.60, High 0.80, Critical 0.92; emits at most one event per level per session.
8. **INV-8** — `Effort.Xhigh` flows to runner via runner-specific knob (e.g. Claude `--reasoning-effort xhigh`); rejected at config parse if runner doesn't support it.
9. **INV-9** — `MaxTurns` cap enforced; agent killed (SIGTERM→SIGKILL) on overrun, exit code mapped to `AgentMaxTurnsExceeded`.
10. **INV-10** — `AllowedFolders` enforced via runner sandbox flag; if runner doesn't support sandboxing, runner emits `AgentSandboxUnsupported` warning event but does NOT silently elevate.
11. **INV-11** — Each runner is registered in DI via `services.AddTransient<IAgentRunner, XxxRunner>()`; factory resolves by `AgentRunnerKind`.
12. **INV-12** — Runner not installed (PATH lookup fail) raises `AgentRunnerNotInstalledException` with the probed path; never silently falls back to a different runner.
