## Goal

`IShellRunner` per §3.10.1 + §3.31.1.4. Runs `bash`, `sh`, `cmd`, `powershell`, and `pwsh` work specs. Implements PS-ISO-1..5 (PowerShell hardening: `-NoProfile -NonInteractive -ExecutionPolicy Bypass -File <script>`, NEVER `-Command "..."`) and CMD-TMP-1 (atomic O_EXCL+0600 / CREATE_NEW + owner-only DACL temp script creation).

## Detailed implementation requirements

### Public API surface

```csharp
namespace AiOrchestrator.Shell;

public sealed class ShellRunner : IShellRunner
{
    public ShellRunner(IProcessSpawner spawner, IFileSystem fs, IClock clock, IEventBus bus, IOptionsMonitor<ShellOptions> opts, ILogger<ShellRunner> logger);
    public ValueTask<ShellRunResult> RunAsync(ShellSpec spec, RunContext ctx, CancellationToken ct);
}

public sealed record ShellOptions { public TimeSpan DefaultTimeout { get; init; } = TimeSpan.FromMinutes(30); public AbsolutePath TempDir { get; init; } = AbsolutePath.Of(System.IO.Path.GetTempPath()); }

public sealed record ShellSpec
{
    public required ShellKind Shell { get; init; }
    public required string Script { get; init; }   // body
    public required AbsolutePath WorkingDirectory { get; init; }
    public required ImmutableDictionary<string, string> Env { get; init; }
    public TimeSpan? Timeout { get; init; }
    public bool CaptureStdoutToLineView { get; init; } = true;
}
public enum ShellKind { Bash, Sh, Cmd, PowerShell, Pwsh }

public sealed record RunContext { public required JobId JobId { get; init; } public required RunId RunId { get; init; } public required AuthContext Principal { get; init; } }
public sealed record ShellRunResult { public required int ExitCode { get; init; } public required TimeSpan Duration { get; init; } public required long StdoutBytes { get; init; } public required long StderrBytes { get; init; } public required bool TimedOut { get; init; } }

internal sealed class SecureTempScript : IAsyncDisposable
{
    public SecureTempScript(IFileSystem fs, AbsolutePath tempDir);
    public ValueTask<AbsolutePath> CreateAsync(ReadOnlyMemory<byte> contents, string fileExtension, CancellationToken ct);
    // Linux/macOS: open() with O_CREAT|O_EXCL|O_WRONLY|O_CLOEXEC, mode 0600
    // Windows: CreateFile with CREATE_NEW + DACL granting only current user FILE_GENERIC_READ|FILE_GENERIC_EXECUTE
    public ValueTask DisposeAsync();   // securely deletes
}

internal sealed class PowerShellCommandLineBuilder
{
    public ImmutableArray<string> Build(AbsolutePath scriptPath);   // -NoProfile, -NonInteractive, -ExecutionPolicy Bypass, -File <path>; NEVER -Command "string"
    public bool ContainsForbiddenFlags(ImmutableArray<string> args);   // -Command, -EncodedCommand are forbidden
}
```

### Behavioral invariants

1. **INV-1** (PS-ISO-1) — PowerShell invocation MUST use `-NoProfile -NonInteractive -ExecutionPolicy Bypass -File <scriptPath>`; the ShellRunner NEVER uses `-Command` or `-EncodedCommand`.
2. **INV-2** (PS-ISO-2) — `-File <scriptPath>` always points at a `SecureTempScript` instance (created via the secure-temp facility, never a user-controlled path).
3. **INV-3** (PS-ISO-3) — Script file extension is `.ps1` (PowerShell) or `.cmd`/`.bat` (cmd) or `.sh` (bash/sh); never executable bit set on Windows; on Unix mode is `0600` (no execute bit).
4. **INV-4** (PS-ISO-4) — `PowerShellCommandLineBuilder.ContainsForbiddenFlags` is checked on every spawn argv as a defense-in-depth assertion; build error if forbidden flags found.
5. **INV-5** (PS-ISO-5) — Environment variables passed via `EnvScope` (job 10), not interpolated into the script.
6. **INV-6** (CMD-TMP-1) — Linux/macOS temp script created with `O_CREAT|O_EXCL|O_WRONLY|O_CLOEXEC, 0600`; existing path causes failure (no overwrite). Windows temp script created with `CREATE_NEW` + DACL granting only the current user read+execute.
7. **INV-7** — Temp script deleted via secure delete on `DisposeAsync` (best-effort overwrite + remove).
8. **INV-8** — Stdout/stderr default-routed to `LineProjector` (job 15) when `CaptureStdoutToLineView = true`.
9. **INV-9** — Timeout default 30 min; on timeout sends SIGTERM, waits 10 s, then SIGKILL (per job 10 contract).
10. **INV-10** — Working directory MUST exist; checked before spawn; throws `WorkingDirectoryNotFoundException`.
