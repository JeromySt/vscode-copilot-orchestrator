# Job 017 — Credential Broker

**Tier:** T4
**Producer id:** `t4-credentials`
**Dependencies:** `t2-process`, `t2-clock`, `t4-audit`
**Estimated LOC:** ~900 production, ~1,300 tests
**Effort:** high (security-critical)

## Goal

`ICredentialBroker` per §3.6 + §3.31.1.1 v1.4 hardenings. Brokers OS-keychain access for git via Git Credential Manager (GCM), enforces URL allowlist, runs the full GCM verb sequence (`get` → `store` | `erase`), uses path-based UDS / named pipes (NOT abstract sockets) with per-message peer-credential check, and engages exponential backoff on credential-invalidation events.

### Public API surface

```csharp
namespace AiOrchestrator.Credentials;

public sealed class CredentialBroker : ICredentialBroker, IAsyncDisposable
{
    public CredentialBroker(IProcessSpawner spawner, IClock clock, IAuditLog audit, IOptionsMonitor<CredentialOptions> opts, ILogger<CredentialBroker> logger);
    public ValueTask<Credential> GetAsync(Uri repoUrl, AuthContext principal, CancellationToken ct);
    public ValueTask StoreAsync(Uri repoUrl, Credential credential, AuthContext principal, CancellationToken ct);
    public ValueTask EraseAsync(Uri repoUrl, AuthContext principal, CancellationToken ct);
}

public sealed record CredentialOptions
{
    public required ImmutableArray<string> AllowedHostSuffixes { get; init; }   // "github.com", "dev.azure.com", "ssh.dev.azure.com" etc.
    public TimeSpan GcmTimeout { get; init; } = TimeSpan.FromSeconds(30);
    public string GcmExecutableName { get; init; } = "git-credential-manager";
    public CredentialBackoffOptions Backoff { get; init; } = new();
}
public sealed record CredentialBackoffOptions { public int FailuresBeforeBackoff { get; init; } = 5; public TimeSpan InitialDelay { get; init; } = TimeSpan.FromMinutes(1); public TimeSpan MaxDelay { get; init; } = TimeSpan.FromHours(1); public double Multiplier { get; init; } = 2.0; }

public sealed record Credential { public required string Username { get; init; } public required ProtectedString Password { get; init; } public required DateTimeOffset RetrievedAt { get; init; } public required string SourceProtocol { get; init; } /* https | ssh | basic */ }

public sealed class GcmInvoker
{
    public GcmInvoker(IProcessSpawner spawner, IOptionsMonitor<CredentialOptions> opts);
    public ValueTask<Credential> GetAsync(Uri repoUrl, CancellationToken ct);          // verb=get, stdin=protocol/host/path, parses response
    public ValueTask StoreAsync(Uri repoUrl, Credential cred, CancellationToken ct);   // verb=store
    public ValueTask EraseAsync(Uri repoUrl, CancellationToken ct);                    // verb=erase
}

public sealed class CredentialIpc : IAsyncDisposable
{
    public CredentialIpc(AbsolutePath socketPath, IProcessSpawner spawner);
    public ValueTask StartListeningAsync(CancellationToken ct);   // POSIX: bind UDS at `socketPath` with 0700 perms; Windows: NamedPipeServerStream with owner-only DACL
    public ValueTask<PeerInfo> GetPeerCredentialsAsync(Stream connection, CancellationToken ct);   // SO_PEERCRED / GetNamedPipeClientProcessId + ImpersonateNamedPipeClient
}
public sealed record PeerInfo { public required int Pid { get; init; } public required uint Uid { get; init; } public required string? UserSid { get; init; } }

internal sealed class HostAllowlistChecker
{
    public HostAllowlistChecker(IOptionsMonitor<CredentialOptions> opts);
    public bool IsAllowed(Uri repoUrl);   // matches host suffix from CredentialOptions.AllowedHostSuffixes
    public CredentialHostNotAllowedException CreateException(Uri repoUrl);
}
public sealed class CredentialHostNotAllowedException : Exception { public Uri Url { get; } public ImmutableArray<string> AllowedSuffixes { get; } }

internal sealed class CredentialBackoffEngine
{
    public CredentialBackoffEngine(IClock clock, IOptionsMonitor<CredentialOptions> opts, IEventBus bus);
    public bool TryEnter(Uri repoUrl, out TimeSpan delayUntil);
    public void RecordFailure(Uri repoUrl);
    public void RecordSuccess(Uri repoUrl);
}
public sealed class CredentialBackoffEngaged { public required Uri RepoUrl { get; init; } public required int FailureCount { get; init; } public required TimeSpan EffectiveDelay { get; init; } public required DateTimeOffset At { get; init; } }
```

### Behavioral invariants

1. **INV-1** (CRED-ACL-1) — `GetAsync` rejects URLs whose host does not match any `AllowedHostSuffixes` with `CredentialHostNotAllowedException`; rejection is audited.
2. **INV-2** (CRED-IPC-1 v1.4) — Linux/macOS: socket path is `/run/ai-orchestrator/cred.<rand>.sock`, NOT abstract socket (`@abstract` namespace); created with 0700 perms in a directory owned by the broker user.
3. **INV-3** (CRED-IPC-2 v1.4) — Per-message peer credentials check on EVERY incoming RPC: Linux uses `SO_PEERCRED` via `getsockopt`; Windows uses `GetNamedPipeClientProcessId` + `ImpersonateNamedPipeClient` to retrieve the SID and re-checks it.
4. **INV-4** (CRED-IPC-3 v1.4) — On any peer-creds mismatch the connection is closed and a `CredentialIpcPeerCredentialMismatch` event is audited.
5. **INV-5** (CRED-VERB-1 v1.4) — Every `GetAsync` call invokes the full `git credential` verb sequence: `git credential fill` → consumer uses cred → `git credential approve` (= `StoreAsync`) on success or `git credential reject` (= `EraseAsync`) on auth failure.
6. **INV-6** (CRED-TIMEOUT-1) — GCM invocation cancels at `CredentialOptions.GcmTimeout` (default 30 s).
7. **INV-7** (CRED-INVAL-1 v1.4) — On `FailuresBeforeBackoff` (default 5) consecutive credential-invalidation events for a URL, broker engages exponential backoff: `InitialDelay * Multiplier^n`, capped at `MaxDelay`.
8. **INV-8** (CRED-INVAL-2 v1.4) — `CredentialBackoffEngaged` event is published exactly once per backoff entry.
9. **INV-9** (CRED-INVAL-3 v1.4) — A successful `GetAsync` resets the failure counter for that URL.
10. **INV-10** — `Credential.Password` is `ProtectedString` (zeroed on dispose); never logged in plaintext.
11. **INV-11** — All operations audited via `IAuditLog.AppendAsync` with PII-redacted URLs (path stripped to host).
12. **INV-12** — Same-uid threat is OUT OF SCOPE; documented in `docs/SECURITY.md` (link in inline comment on `CredentialIpc`).
