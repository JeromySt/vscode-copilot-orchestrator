# Job 020 — Hook Gate Daemon

**Tier:** T4
**Producer id:** `t4-hook-gate`
**Dependencies:** `t2-process`, `t2-filesystem`, `t2-clock`, `t3-event-bus`, `t4-credentials`
**Estimated LOC:** ~1,200 production, ~1,600 tests
**Effort:** high (security-critical)

## Goal

Out-of-process daemon RPC for hook gating per §3.31.1.0 + §3.8 v1.4. Daemon owns nonce/SID issuance, receives hook check-in, validates, and issues an unforgeable approval token. Implements HK-GATE-LINK-* v1.4 (PRIMARY: bind-mount/junction redirection — works without privilege; BEST-EFFORT: chattr +i / chflags / DACL-deny — emits `HookGateNonceImmutabilityUnsupported` on no-op), nonce rotation ≤ 5 min, st_nlink check (Unix), reparse-point check (Windows).

### Behavioral invariants

1. **INV-1** — Daemon runs out-of-process; client (`IHookGateClient`) communicates over UDS (Linux/macOS) / NamedPipe (Windows) with **per-message peer-credential check** (same pattern as job 17 CRED-IPC-2).
2. **INV-2** (HK-GATE-LINK-1 v1.4) — `IRedirectionManager.InstallRedirectionAsync` MUST first try bind-mount (Linux/macOS) / junction (Windows) — these work in unprivileged user contexts (with `fusermount`/Developer Mode). Symlink fallback only used if both fail and ALWAYS emits `HookGateNonceImmutabilityUnsupported`.
3. **INV-3** (HK-GATE-LINK-2 v1.4) — Best-effort immutability: daemon ATTEMPTS `chattr +i` (Linux), `chflags uchg` (macOS), DACL-deny (Windows). On no-op (e.g. unprivileged), it emits `HookGateNonceImmutabilityUnsupported { Path, Mechanism, Reason }`. If `Options.RequireImmutability = true`, daemon ALSO logs at warning level.
4. **INV-4** (HK-GATE-LINK-3 v1.4) — `LinkValidator.ValidateAsync` rejects the check-in if: (a) Linux: `st_nlink > 1` (multiple hardlinks indicate tamper) OR resolved canonical path is outside the worktree; (b) Windows: file is a reparse point with target outside the worktree.
5. **INV-5** — Nonce rotation is automatic every `Options.NonceRotation` (default 5 min); old nonce remains valid for one rotation period (overlap window).
6. **INV-6** — Approval HMAC = HMAC-SHA256(currentNonce, canonicalForm(request)); `ApprovalTokenTtl` default 2 min.
7. **INV-7** — Denied check-in raises `HookApprovalDeniedException` with `Reason` and is audited via `IAuditLog`.
8. **INV-8** — Daemon refuses to start if `SocketPath` parent directory is not owned by the broker user OR has perms broader than 0700.
9. **INV-9** — Daemon supports graceful shutdown (`StopAsync`); in-flight check-ins complete; new ones are rejected with `Reason = "daemon shutting down"`.
10. **INV-10** — All cryptographic ops use `System.Security.Cryptography.HMACSHA256`.
