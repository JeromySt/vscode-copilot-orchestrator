# Job 018 — Git Operations

**Tier:** T4
**Producer id:** `t4-git`
**Dependencies:** `t2-process`, `t2-filesystem`, `t4-credentials`
**Estimated LOC:** ~1,400 production, ~1,800 tests

## Goal

`IGitOperations` over **LibGit2Sharp** for the structured-data path (status, branches, commits, refs, walk) and a small `GitShellInvoker` over the `git` CLI for operations LibGit2Sharp cannot do safely (worktrees, partial clone, commit-graph, sparse-checkout, fsmonitor handoff). Implements LG2-CANCEL-* (cooperative cancellation through progress callbacks) and LG2-BRK-* (typed exception mapping).

### Public API surface

```csharp
namespace AiOrchestrator.Git;

public sealed class GitOperations : IGitOperations, IAsyncDisposable
{
    public GitOperations(ICredentialBroker creds, IProcessSpawner spawner, IFileSystem fs, IClock clock, IOptionsMonitor<GitOptions> opts, ILogger<GitOperations> logger);
    public ValueTask<Repository> OpenAsync(AbsolutePath path, CancellationToken ct);
    public ValueTask<CloneResult> CloneAsync(CloneRequest request, CancellationToken ct);
    public ValueTask<FetchResult> FetchAsync(Repository repo, FetchRequest request, CancellationToken ct);
    public ValueTask<MergeResult> MergeAsync(Repository repo, MergeRequest request, CancellationToken ct);
    public ValueTask<PushResult> PushAsync(Repository repo, PushRequest request, CancellationToken ct);
    public ValueTask<WorktreeHandle> AddWorktreeAsync(Repository repo, WorktreeRequest request, CancellationToken ct);
    public ValueTask RemoveWorktreeAsync(WorktreeHandle handle, bool force, CancellationToken ct);
    public ValueTask<CommitInfo> CreateCommitAsync(Repository repo, CommitRequest request, CancellationToken ct);
    public ValueTask<RefUpdate> UpdateRefAsync(Repository repo, string refName, CommitSha newTarget, CommitSha? oldTarget, CancellationToken ct);
    public IAsyncEnumerable<CommitInfo> WalkAsync(Repository repo, WalkRequest request, CancellationToken ct);
    public ValueTask<DiffResult> DiffAsync(Repository repo, DiffRequest request, CancellationToken ct);
}

public sealed record GitOptions { public TimeSpan ProgressTickInterval { get; init; } = TimeSpan.FromMilliseconds(100); public AbsolutePath? GitExecutable { get; init; } public bool PreferShellForWorktree { get; init; } = true; }

internal sealed class LibGit2Bridge
{
    public LibGit2Bridge(ICredentialBroker creds);
    // Wraps LG2 calls, maps progress callbacks to ct.IsCancellationRequested
    // Maps LG2 exceptions to typed: MergeConflictException, RemoteRejectedException, WorktreeLockedException, RefUpdateRaceException, AuthFailureException, NetworkErrorException, RefNotFoundException
}

public abstract class GitOperationException : Exception { protected GitOperationException(string msg, Exception? inner = null) : base(msg, inner) { } }
public sealed class MergeConflictException : GitOperationException { public required ImmutableArray<RepoRelativePath> ConflictingPaths { get; init; } }
public sealed class RemoteRejectedException : GitOperationException { public required string Reason { get; init; } public required string RemoteUrl { get; init; } }
public sealed class WorktreeLockedException : GitOperationException { public required AbsolutePath WorktreePath { get; init; } public required string LockReason { get; init; } }
public sealed class RefUpdateRaceException : GitOperationException { public required string RefName { get; init; } public required CommitSha ExpectedOld { get; init; } public required CommitSha ActualOld { get; init; } }
public sealed class AuthFailureException : GitOperationException { public required Uri RepoUrl { get; init; } }
public sealed class NetworkErrorException : GitOperationException { public required Uri RepoUrl { get; init; } public required bool IsRetryable { get; init; } }
public sealed class RefNotFoundException : GitOperationException { public required string RefName { get; init; } }

internal sealed class GitShellInvoker
{
    public GitShellInvoker(IProcessSpawner spawner, IOptionsMonitor<GitOptions> opts);
    public ValueTask<ShellResult> RunAsync(GitVerb verb, ImmutableArray<string> args, AbsolutePath workingDir, CancellationToken ct);
}
public enum GitVerb { Worktree, SparseCheckout, CommitGraph, FsMonitor, MaintenanceRun }   // ALLOWLIST — no other verbs may be invoked through the shell
```

### Behavioral invariants

1. **INV-1** (LG2-CANCEL-1) — Every long-running LG2 op (clone, fetch, push, merge, walk) registers a progress callback that returns `false` (i.e. abort) when `CancellationToken.IsCancellationRequested`.
2. **INV-2** (LG2-CANCEL-2) — Cancellation of an in-progress operation completes within `2 * ProgressTickInterval` (default 200 ms) on the test rig.
3. **INV-3** (LG2-BRK-1..N) — Every raw LG2 exception is caught at the bridge boundary and re-thrown as a typed `GitOperationException` subclass.
4. **INV-4** — `GitShellInvoker.RunAsync` rejects any verb not in the `GitVerb` allowlist enum at compile time.
5. **INV-5** — Worktree add/remove uses `GitShellInvoker` with `--lock` semantics; `WorktreeHandle.Dispose` removes the worktree.
6. **INV-6** — `UpdateRefAsync` enforces optimistic CAS (`oldTarget != null`); mismatch raises `RefUpdateRaceException`.
7. **INV-7** — `CreateCommitAsync` writes commit with `committer` populated from `IClock.UtcNow` (no implicit `DateTime.UtcNow`).
8. **INV-8** — Credential callback delegates to `ICredentialBroker.GetAsync(repoUrl, principal, ct)`.
9. **INV-9** — `CloneRequest.IsBare` defaults `false`; `Filter` (e.g. `blob:none`) flows through shell when LG2 lacks support.
10. **INV-10** — All exception messages are PII-safe (no credentials in `Message` or `InnerException.Message`).
