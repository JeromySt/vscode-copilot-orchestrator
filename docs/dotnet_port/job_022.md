# Job 022 — Concurrency: Host Broker

**Tier:** T5
**Producer id:** `t5-conc-broker`
**Dependencies:** `t2-process`, `t2-clock`, `t3-event-bus`, `t5-conc-user`
**Estimated LOC:** ~1,100 production, ~1,500 tests

## Goal

Host-wide concurrency broker per §3.9.2 + §3.31.1.0 v1.4. Out-of-process daemon arbitrates concurrency across multiple AIO clients on the same host with proportional fairness OR strict round-robin between users. Implements CONC-BROKER-2/3 v1.4 (path-based UDS at `/run/ai-orchestrator/coord.sock`, NOT abstract socket; per-message peer-creds), `ConcurrencyHostQueued` event with `actionableHint` + `etaSeconds`.

## Source-doc sections

- §3.31.1.0 CONC-BROKER-2/3 v1.4 (path-based UDS, peer-creds)

## Detailed implementation requirements

### Public API surface

```csharp
namespace AiOrchestrator.Concurrency.Broker;

public sealed class HostConcurrencyBrokerDaemon : IHostedService, IAsyncDisposable
{
    public HostConcurrencyBrokerDaemon(IRpcServer rpc, FairnessScheduler scheduler, IClock clock, IEventBus bus, IOptionsMonitor<BrokerOptions> opts, ILogger<HostConcurrencyBrokerDaemon> logger);
    public Task StartAsync(CancellationToken ct);
    public Task StopAsync(CancellationToken ct);
}

public sealed record BrokerOptions
{
    public AbsolutePath SocketPath { get; init; } = AbsolutePath.Of("/run/ai-orchestrator/coord.sock");
    public string PipeName { get; init; } = @"\\.\pipe\AiOrchestratorCoord";
    [Range(1, 1024)] public int MaxConcurrentHostWide { get; init; } = 16;
    public HostFairness HostFairness { get; init; } = HostFairness.Proportional;
}
public enum HostFairness { Proportional, StrictRoundRobin }

public interface IHostConcurrencyBrokerClient
{
    ValueTask<HostAdmission> AcquireAsync(AuthContext principal, JobId job, CancellationToken ct);
}
public sealed class HostAdmission : IAsyncDisposable
{
    public required AuthContext Principal { get; init; }
    public required JobId JobId { get; init; }
    public required DateTimeOffset AdmittedAt { get; init; }
    public required string BrokerLeaseId { get; init; }
    public ValueTask DisposeAsync();   // sends release to broker
}

internal sealed class FairnessScheduler
{
    public FairnessScheduler(IClock clock, IOptionsMonitor<BrokerOptions> opts, IEventBus bus);
    public ValueTask<FairnessDecision> EnqueueAsync(AuthContext principal, JobId job, CancellationToken ct);
    // Proportional: weights by recent admission count to give each principal a fair share
    // StrictRoundRobin: cycles through principals deterministically
}
public sealed record FairnessDecision { public required string LeaseId { get; init; } public required TimeSpan EstimatedWait { get; init; } public required int QueuePosition { get; init; } }

public sealed class ConcurrencyHostQueued
{
    public required AuthContext Principal { get; init; }
    public required JobId JobId { get; init; }
    public required int QueuePosition { get; init; }
    public required TimeSpan EtaSeconds { get; init; }   // CONC-BROKER hint
    public required string ActionableHint { get; init; }   // e.g. "5 jobs ahead from same user, consider raising MaxConcurrentPerUser"
    public required DateTimeOffset At { get; init; }
}

public sealed class BrokerUnavailableException : Exception { public required AbsolutePath SocketPath { get; init; } }
```

### Behavioral invariants

1. **INV-1** (CONC-BROKER-1) — Single daemon owns the global semaphore; clients obtain leases via RPC.
2. **INV-2** (CONC-BROKER-2 v1.4) — Linux/macOS UDS at `/run/ai-orchestrator/coord.sock` (path-based, NOT `@abstract`), 0700 perms in directory owned by daemon user.
3. **INV-3** (CONC-BROKER-3 v1.4) — Per-message peer-creds check on every RPC (`SO_PEERCRED` / `GetNamedPipeClientProcessId` + `ImpersonateNamedPipeClient`); mismatch closes connection and audits.
4. **INV-4** (CONC-BROKER-FAIR-1) — `HostFairness.Proportional` divides the active slot pool roughly proportionally to active principals' weights (default uniform weight 1).
5. **INV-5** (CONC-BROKER-FAIR-2) — `HostFairness.StrictRoundRobin` admits in deterministic round-robin order across principals; admission within a principal is FIFO.
6. **INV-6** — `ConcurrencyHostQueued` is emitted with non-empty `ActionableHint` and `EtaSeconds` computed from observed admission rate over a 30 s window.
7. **INV-7** — Lease has TTL default 5 min; lease expiry frees the slot and emits `HostAdmissionExpired`.
8. **INV-8** — Daemon graceful shutdown: drains in-flight admissions, refuses new ones with `BrokerUnavailableException`.
9. **INV-9** — Client `IHostConcurrencyBrokerClient` falls back to "no host fairness" mode (passes through to per-user limiter only) if the broker is not running, with a single warning log.
