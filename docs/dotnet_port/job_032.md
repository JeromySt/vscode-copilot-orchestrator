# Job 032 — Hosting

## Goal

`AiOrchestrator.Hosting` per §3.13. Wraps `Microsoft.Extensions.Hosting` and exposes `IHostBuilderExtensions.AddAiOrchestrator()` that calls every `CompositionRoot.AddXxx` extension in dependency order, registers `IHostedService` instances (HookGate daemon, Concurrency broker, Plan scheduler), wires logger + telemetry, and provides graceful shutdown.

## Detailed implementation requirements

### Public API surface

```csharp
namespace AiOrchestrator.Hosting;

public static class HostBuilderExtensions
{
    public static IHostBuilder UseAiOrchestrator(this IHostBuilder builder, Action<AiOrchestratorOptions>? configure = null);
    public static IServiceCollection AddAiOrchestrator(this IServiceCollection services, IConfiguration config);
}

public sealed record AiOrchestratorOptions
{
    public AbsolutePath StoreRoot { get; init; } = AbsolutePath.Of(System.IO.Path.Combine(Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData), "ai-orchestrator"));
    public bool EnableHookGate { get; init; } = true;
    public bool EnableConcurrencyBroker { get; init; } = true;
    public bool EnablePluginLoader { get; init; } = true;
    public bool EnableTelemetry { get; init; } = false;
}

public sealed class AiOrchestratorRoot : IAsyncDisposable
{
    public AiOrchestratorRoot(IServiceProvider sp);
    public IServiceProvider Services { get; }
    public ValueTask DisposeAsync();   // graceful shutdown of all hosted services in reverse order
}
```

### Behavioral invariants

1. **INV-1** (H-1) — `AddAiOrchestrator` calls every `CompositionRoot.AddXxx` extension exactly once; no double registration.
2. **INV-2** (H-2) — Composition root is the ONLY place that performs `services.AddSingleton<IFoo, Foo>()` for AIO services; other layers reference interfaces only (enforced by analyzer OE0001).
3. **INV-3** (H-3) — Hosted services start in dependency order: HookGate → Broker → Scheduler.
4. **INV-4** (H-4) — Hosted services stop in reverse order with graceful shutdown timeout (default 30 s).
5. **INV-5** (H-5) — `AiOrchestratorOptions.Enable*` flags toggle component registration; disabled components do NOT register.
6. **INV-6** (H-6) — Configuration is loaded via `LayeredConfigurationProvider` (job 005) BEFORE composition root is built.
