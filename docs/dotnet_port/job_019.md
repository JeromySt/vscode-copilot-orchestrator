# Job 019 — Worktree Lease

**Tier:** T4
**Producer id:** `t4-worktree-lease`
**Dependencies:** `t2-filesystem`, `t2-clock`, `t3-event-bus`, `t4-git`
**Estimated LOC:** ~600 production, ~900 tests

## Goal

`IWorktreeLease` per §3.31.2.1 — CAS-under-lock acquisition of an exclusive lease on a git worktree, with a monotonic `FencingToken` enforced by storage. Implements LS-CAS-* (atomic compare-and-swap on `<worktree>/.aio/lease.json` with `FileShare.None`), LS-INF-* (storage rejects writes with stale `FencingToken`), and `WorktreeLeaseStolen` event detection.

### Behavioral invariants

1. **INV-1** (LS-CAS-1) — Lease acquisition opens `<worktree>/.aio/lease.json` with `FileShare.None`; under that exclusive lock, reads current `FencingToken`, computes `next = current + 1`, and writes the new content; release removes the file.
2. **INV-2** (LS-CAS-2) — On contention (`IOException` from `FileShare.None`), retries with `AcquireRetryDelay` until `AcquireTimeout`; throws `LeaseAcquireTimeoutException` after timeout.
3. **INV-3** (LS-CAS-3) — `FencingToken` is strictly monotonic: starting from `1`, `next = current + 1` always; never reused.
4. **INV-4** (LS-INF-1) — Any write performed via `CasLeaseStore.EnforceWriteWithTokenAsync` checks the stored lease token against the supplied token; mismatch throws `StaleLeaseTokenException` and the write is NOT performed.
5. **INV-5** (LS-INF-2) — All worktree-mutating operations in higher layers (commit phase, work phase, snapshot validation) MUST flow through `EnforceWriteWithTokenAsync`; enforced by analyzer (job 011).
6. **INV-6** — `StaleLeaseDetector` polls every `StaleCheckInterval`; if the lease file's token != the holder's expected token, emits `WorktreeLeaseStolen` event.
7. **INV-7** — `LeaseHandle.DisposeAsync` is idempotent.
8. **INV-8** — Lease file format = JSON with `SchemaVersion = "1"`; reader rejects unknown versions with `UnsupportedLeaseSchemaException`.
9. **INV-9** — Renewal rewrites the lease file with extended `ExpiresAt` AND increments the token (every renewal is a fresh CAS).
10. **INV-10** — Read-only inspect (`InspectAsync`) opens with `FileShare.Read` and returns the parsed `LeaseInfo`.
