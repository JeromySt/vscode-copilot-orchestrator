### Behavioral invariants

1. **INV-1** — Project contains only `interface`, `enum`, `readonly record struct`, and `sealed record` declarations; zero `class` declarations.
2. **INV-2** — Every async-returning method has a `CancellationToken` as the last parameter.
3. **INV-3** — Every public member has an XML doc comment (SA1600 = error).
4. **INV-4** — Single project reference: `AiOrchestrator.Models`.
5. **INV-5** — No reference to `Microsoft.Extensions.*` (host concern).
6. **INV-6** — Every interface returns `ValueTask`/`ValueTask<T>`/`Task<T>` or `IAsyncEnumerable<T>`; no raw `Task` (fire-and-forget banned).
7. **INV-7** — Every interface that owns a resource (`IProcessHandle`, `Credential`) implements `IAsyncDisposable` or `IDisposable`.
