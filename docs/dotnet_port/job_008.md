# Job 008 — Clock

**Tier:** T2

**Dependencies:** `t1-abstractions`, `t1-testkit`
**Estimated LOC:** ~250 production, ~400 tests

## Goal

Implement `AiOrchestrator.Time` — `SystemClock` (wraps `TimeProvider.System`), `MonotonicGuard` (panics if monotonic clock regresses — VM/hardware bug surface), `SystemDelayProvider` (wraps `Task.Delay`). All time-driven behavior in the system flows through these abstractions; **zero `DateTime.UtcNow`/`Environment.TickCount`/`Thread.Sleep` in production code** (banned by `build/banned.txt` from job 00).

## Detailed implementation requirements

### Public API surface

```csharp
namespace AiOrchestrator.Time;

public sealed class SystemClock : IClock
{
    public SystemClock(TimeProvider? underlying = null);
    public DateTimeOffset UtcNow => _underlying.GetUtcNow();
    public long MonotonicMilliseconds => _underlying.GetTimestamp() / (Stopwatch.Frequency / 1000);
}

public sealed class MonotonicGuard : IClock
{
    public MonotonicGuard(IClock inner, ITelemetrySink telemetry);
    public DateTimeOffset UtcNow => _inner.UtcNow;
    public long MonotonicMilliseconds // wraps inner; throws MonotonicRegressionException if value < last observed
}

public sealed class MonotonicRegressionException : Exception { public long Previous { get; } public long Current { get; } }

public sealed class SystemDelayProvider : IDelayProvider
{
    public SystemDelayProvider(TimeProvider? underlying = null);
    public Task Delay(TimeSpan delay, CancellationToken ct);
}

namespace AiOrchestrator.Composition;
public static partial class CompositionRoot { public static IServiceCollection AddTime(this IServiceCollection services); }
```

### Behavioral invariants

1. **INV-1** — `SystemClock.UtcNow` returns UTC `DateTimeOffset` (`Offset == TimeSpan.Zero`).
2. **INV-2** — `SystemClock.MonotonicMilliseconds` is non-decreasing within a single process — verified by `CLOCK-2`.
3. **INV-3** — `MonotonicGuard` records `Previous` and throws `MonotonicRegressionException` and emits `MonotonicClockRegression` telemetry counter on regression.
4. **INV-4** — `SystemDelayProvider.Delay` honors cancellation in ≤ 50 ms.
5. **INV-5** — Composition root registers `SystemClock` wrapped in `MonotonicGuard` by default.
6. **INV-6** — Project has zero references to `Microsoft.Extensions.*` except `Logging.Abstractions` (for `ITelemetrySink` injection chain).
