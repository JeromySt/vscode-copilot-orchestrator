# Job 026 — Plugin Loader

**Tier:** T6
**Producer id:** `t6-plugins`
**Dependencies:** `t2-filesystem`, `t2-clock`, `t3-event-bus`, `t4-audit`
**Estimated LOC:** ~1,000 production, ~1,400 tests
**Effort:** high (security-critical)

## Goal

Plugin loading per §3.10.2 + §3.31.1.5 — load third-party `IAgentRunner` / `IShellRunner` / `ITool` plugins from a configured plugin directory using `AssemblyLoadContext` isolation. Implements TRUST-ACL-* (trust-file gating with strict perms), `MinHostVersion`/`MaxHostVersion` compat, and `[PluginCapability]` attribute-based capability sandboxing.

### Public API surface

```csharp
namespace AiOrchestrator.Plugins;

public sealed class PluginLoader : IPluginLoader, IAsyncDisposable
{
    public PluginLoader(IFileSystem fs, IClock clock, IEventBus bus, IAuditLog audit, IOptionsMonitor<PluginOptions> opts, ILogger<PluginLoader> logger);
    public ValueTask<ImmutableArray<LoadedPlugin>> DiscoverAndLoadAsync(CancellationToken ct);
    public ValueTask UnloadAsync(LoadedPlugin plugin, CancellationToken ct);
    public IReadOnlyList<LoadedPlugin> Loaded { get; }
}

public sealed record PluginOptions { public required AbsolutePath PluginRoot { get; init; } public required AbsolutePath TrustFilePath { get; init; } public bool RequireTrustFile { get; init; } = true; public Version HostVersion { get; init; } = typeof(PluginLoader).Assembly.GetName().Version!; }

public sealed record LoadedPlugin { public required string PluginId { get; init; } public required Version PluginVersion { get; init; } public required AbsolutePath AssemblyPath { get; init; } public required ImmutableArray<PluginCapability> Capabilities { get; init; } public required ImmutableArray<Type> ExportedTypes { get; init; } }

public sealed record PluginManifest { public required string PluginId { get; init; } public required Version PluginVersion { get; init; } public required Version MinHostVersion { get; init; } public required Version MaxHostVersion { get; init; } public required ImmutableArray<PluginCapability> Capabilities { get; init; } public required string AssemblyFileName { get; init; } public required string AuthorPublicKeyFingerprint { get; init; } }

public sealed record TrustFile { public required ImmutableArray<TrustedPlugin> TrustedPlugins { get; init; } public required DateTimeOffset SignedAt { get; init; } public required byte[] Ed25519Signature { get; init; } public required string SignerKeyId { get; init; } }
public sealed record TrustedPlugin { public required string PluginId { get; init; } public required string AssemblySha256 { get; init; } public required string AuthorPublicKeyFingerprint { get; init; } public required Version MaxAcceptableVersion { get; init; } }

[AttributeUsage(AttributeTargets.Class, AllowMultiple = true)]
public sealed class PluginCapabilityAttribute : Attribute { public PluginCapability Capability { get; } public PluginCapabilityAttribute(PluginCapability capability); }
public enum PluginCapability { ReadFiles, WriteFiles, RunProcesses, NetworkAccess, ReadAuditLog, WriteAuditLog, AccessCredentials }

internal sealed class PluginLoadContext : System.Runtime.Loader.AssemblyLoadContext
{
    public PluginLoadContext(AbsolutePath assemblyPath, bool isCollectible) : base(name: $"plugin:{assemblyPath.Value}", isCollectible: isCollectible) { }
    protected override Assembly? Load(AssemblyName assemblyName);   // resolves dependencies from plugin directory; never from default ALC
}

internal sealed class TrustFileVerifier
{
    public TrustFileVerifier(IFileSystem fs);
    public ValueTask<bool> IsTrustFileValidAsync(AbsolutePath trustFile, CancellationToken ct);   // perms 0600 / owner-only DACL; valid Ed25519 sig
    public ValueTask<TrustFile> LoadAndVerifyAsync(AbsolutePath trustFile, byte[] expectedSignerPubKey, CancellationToken ct);
}

internal sealed class CapabilityChecker
{
    public bool IsAllowed(LoadedPlugin plugin, PluginCapability capability);
    public ValueTask EnforceAtCallSiteAsync(PluginCapability required, [System.Runtime.CompilerServices.CallerMemberName] string? caller = null);   // throws PluginCapabilityDeniedException
}
public sealed class PluginCapabilityDeniedException : Exception { public required string PluginId { get; init; } public required PluginCapability Required { get; init; } }
```

### Behavioral invariants

1. **INV-1** (TRUST-ACL-1) — Trust file MUST exist and have perms `0600` (Unix) / owner-only DACL (Windows); broader perms cause a startup error and emit `PluginTrustFileInvalidPerms`.
2. **INV-2** (TRUST-ACL-2) — Trust file MUST be Ed25519-signed by an authorized signer key (configured via `Options`); invalid signature → startup error.
3. **INV-3** (TRUST-ACL-3) — A discovered plugin MUST be listed in the trust file with a SHA-256 hash that matches the on-disk assembly; mismatch → plugin REJECTED, audit event emitted.
4. **INV-4** — Plugin manifest is `manifest.json` next to the assembly; missing or malformed → plugin rejected.
5. **INV-5** — `MinHostVersion <= HostVersion <= MaxHostVersion`; mismatch → rejected with audit event `PluginCompatRejected`.
6. **INV-6** — Each plugin loaded into its own `AssemblyLoadContext` with `isCollectible: true`; `UnloadAsync` allows GC reclaim.
7. **INV-7** — Plugin types declare `[PluginCapability(...)]`; the loader records declared capabilities and refuses load if a required capability is not declared at type level.
8. **INV-8** — Any host code that performs a capability-bound action (file write outside plugin sandbox, process spawn, network call, credential read) MUST call `CapabilityChecker.EnforceAtCallSiteAsync(...)`; analyzer enforces.
9. **INV-9** — Plugin discovery audit: `PluginDiscovered`, `PluginLoaded`, `PluginRejected`, `PluginUnloaded` events emitted with plugin id + assembly hash.
10. **INV-10** — `RequireTrustFile = false` is allowed only for development; emits `PluginUnverifiedLoadAllowed` warning event on every load.
