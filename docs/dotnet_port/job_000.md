### Post-check helper scripts (PowerShell, all under `scripts/dotnet/`)

Every script accepts `-Project <name>` and exits 0/1. All scripts MUST be invocable on PS 5.1 + PS 7.

#### `scripts/dotnet/check-coverage.ps1`
- Runs `dotnet test --collect:"XPlat Code Coverage" --results-directory $env:TEMP/aio-cov-<job>` if a fresh `coverage.cobertura.xml` is not already present.
- Parses cobertura xml; computes line + branch rate for files matching `src/dotnet/<Project>/**`.
- Fails if line < `$MinLine` or branch < `$MinBranch`. Writes JSON report.

#### `scripts/dotnet/check-analyzers.ps1`
- Runs `dotnet build` with `/p:ReportAnalyzer=true /p:WarningsAsErrors=true /v:n` redirecting to `$env:TEMP/aio-build-<job>.log`.
- Greps for `\bOE0\d{4}\b` in the touched project subtree; exits 1 if any matches.

#### `scripts/dotnet/check-banned-apis.ps1`
- Reads `build/banned.txt`.
- For each pattern, runs `Select-String` against `src/dotnet/<Project>/**/*.cs`; exits 1 on any match outside the explicitly-allowed owning project (e.g., `Process.Start` allowed only inside `AiOrchestrator.Process`).

#### `scripts/dotnet/check-contract-tests.ps1`
- Parses the job spec markdown for the `## Acceptance tests` section.
- Greps `tests/dotnet/<Project>.Tests/**/*.cs` for each test name; verifies presence + `[ContractTest(...)]` attribute on the method.
- Exits 1 if any are missing.

#### `scripts/dotnet/check-public-api.ps1`
- Uses `Microsoft.CodeAnalysis.PublicApiAnalyzers` baseline files (`PublicAPI.Shipped.txt`, `PublicAPI.Unshipped.txt`).
- Diffs the touched project's unshipped API list against the "Public API surface" section of the job spec; exits 1 on drift.

#### `scripts/dotnet/check-composition.ps1`
- For every `internal sealed class` or `public sealed class` in `src/dotnet/<Project>/**/*.cs` that implements an interface declared in `AiOrchestrator.Abstractions`, verifies a registration exists in any partial `CompositionRoot.*.cs`.
- Exits 1 on missing registration.

#### `scripts/dotnet/run-all-postchecks.ps1`
- Orchestrates PC-1..PC-9 in order; aggregates JSON reports; produces a single summary JSON consumed by the orchestrator.

### Behavioral invariants

1. **INV-1** ŌĆö `dotnet build src/dotnet/AiOrchestrator.sln -warnaserror` exits 0 on a fresh checkout.
2. **INV-2** ŌĆö `dotnet restore src/dotnet/AiOrchestrator.sln --locked-mode` succeeds; `packages.lock.json` is committed for every project.
3. **INV-3** ŌĆö Zero `<PackageReference ... Version="...">` entries anywhere; CPM is the only version source.
4. **INV-4** ŌĆö `<NoWarn>` is empty in every csproj.
5. **INV-5** ŌĆö Every helper script in `scripts/dotnet/` is idempotent and read-only against `src/`.
