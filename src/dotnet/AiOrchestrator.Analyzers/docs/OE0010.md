# OE0010 — Banned time/clock API

**Severity:** Error  
**Category:** AiOrchestrator  

## Rule description

The following APIs are banned because they make code non-deterministic and hard to test:

| Banned API | Replacement |
|---|---|
| `DateTime.UtcNow` | `IClock.UtcNow` |
| `DateTime.Now` | `IClock.UtcNow` |
| `Environment.TickCount` / `TickCount64` | `IClock.MonotonicMilliseconds` |
| `Thread.Sleep(int)` | `IDelayProvider.DelayAsync(TimeSpan, CancellationToken)` |

Injecting `IClock` / `IDelayProvider` allows tests to control time.

## Non-compliant code

```csharp
// ❌ Non-testable time access
public class Scheduler
{
    public bool IsExpired(DateTimeOffset due)
        => DateTime.UtcNow > due; // OE0010
}
```

## Compliant code

```csharp
// ✅ Inject IClock
public class Scheduler
{
    private readonly IClock _clock;
    public Scheduler(IClock clock) => _clock = clock;
    public bool IsExpired(DateTimeOffset due) => _clock.UtcNow > due;
}
```
